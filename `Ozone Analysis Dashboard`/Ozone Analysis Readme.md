# 🌍 Ozone Analysis Dashboard

## 📌 Project Overview
The Ozone Analysis Dashboard is an interactive Tableau project designed to analyze ozone trends across time, locations, and atmospheric pressure levels. 

This dashboard helps identify:
- Long-term ozone trends
- Site-wise ozone comparison
- Ozone variation by atmospheric pressure
- Geographic distribution of ozone monitoring sites

---

## 🎯 Objectives
- Analyze ozone concentration trends over time
- Compare ozone levels across monitoring sites
- Study ozone behavior across pressure levels
- Visualize geographic distribution using maps

---

## 📊 Dashboard Components

### 1️⃣ Ozone Trend Over Time
- Line chart showing ozone variation from 1994–2022
- Helps identify patterns, fluctuations, and stability trends

### 2️⃣ Site Comparison
- Horizontal bar chart
- Compares ozone levels across 21 monitoring sites
- Identifies highest and lowest ozone concentration areas

### 3️⃣ Ozone by Pressure
- Line chart visualizing ozone concentration against atmospheric pressure
- Shows inverse relationship between ozone and pressure

### 4️⃣ Site Map
- Geographical map displaying site locations
- Enables spatial analysis of ozone levels
